# Infinity Electronics

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-org/infinity-electronics.git
cd infinity-electronics
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

---

## Storybook

To view component documentation:

```bash
npm run storybook
```

Open [http://localhost:6006](http://localhost:6006) in your browser.

---

## Build for production

```bash
npm run build
```

---

## Troubleshooting

- Make sure you have Node.js v18 or v20 installed.
- If you see errors about missing dependencies, run `npm install` again.
- For Tailwind CSS and Storybook issues, ensure you have `postcss`, `autoprefixer`, and `tailwindcss` installed.

---

## Project Structure

- `app/` — Nuxt app source code
- `app/components/` — Vue components
- `app/pages/` — Nuxt pages
- `app/types/` — TypeScript types
- `.storybook/` — Storybook configuration
